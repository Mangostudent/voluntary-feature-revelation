FIGURES DIRECTORY

Place all charts, plots, architectural diagrams, and illustrations here.

Tips for the paper:
- Format: Prefer vector formats like .pdf or .eps for sharp, high-quality printing. You can also use high-resolution .png or .jpg (minimum 300 DPI, 600 DPI for line art).
- Naming: Save images with simple names (e.g., architecture.pdf, results_chart.pdf) matching the \includegraphics{...} statements in your LaTeX files.
- Graphicspath: The main LaTeX file is configured with \graphicspath{{figures/}}, so you do not need to prepend "figures/" in your \includegraphics{...} commands.
